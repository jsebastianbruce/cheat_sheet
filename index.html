TABLE CSS PROPETIES

Property Function
color Specifies the color of text
text-align Specifies the horizontal alignment of text in an element.
text-decoration Specifies the decoration added to text.
text-transform Controls the capitalization of text.
line-height Specifies the height of a line.
letter-spacing Increases or decreases the space between characters in a text.
font-family Specifies the font for an element.
font-size Sets the size of a font.
font-style Specifies the font style for a text.
font-weight Sets how thick or thin characters in text should be displayed.

TABLE CSS VALUES

Property Function
color white, black, red, blue, yellow, green, etc
text-align left, right, center, justify
text-decoration overline, line-through, underline
text-transform none, capitalize, uppercase, lowercase, initial, inherit
line-height normal, a number, a length, percentage, initial, inherit
letter-spacing normal, a length, initial, inherit
font-family verdana, arial, courier, etc
font-size medium, xx-small, x-small, small, large, x-large, etc
font-style normal, italic, oblique, initial, inherit
font-weight normal, bold, bolder, lighter, a number between 100 and 900
